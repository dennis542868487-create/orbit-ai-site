# Orbit AI Website

Recommended structure: 5 pages.

- `index.html` - main landing page
- `pricing.html` - package comparison
- `ai-updates.html` - AI-powered update service
- `process.html` - demo-first workflow and payment flow
- `contact.html` - phone, email, and request form

The site is static and can be opened directly in a browser. It uses the provided Orbit AI flyer as a local asset at `assets/orbit-ai-services.jpg`.
